# Delete Media API

## Problem

A media file exists in two places:
- Supabase Storage
- MongoDB Metadata

Deleting only one creates inconsistent data.

## Solution

1. Verify media exists.
2. Verify ownership.
3. Delete from Supabase.
4. Delete from MongoDB.
5. Return success.

## Lessons Learned

- Difference between metadata and storage.
- Why operation order matters.
- Importance of authorization.
- Orphan files.
- Consistency across systems.
